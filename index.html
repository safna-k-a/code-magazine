The Basic Language of the Web:HTML

Pssted by Laura Johns on monday, June 21st 2027

All modern websites and web applications are built using three fundamental technologies: HTML,CSS and JavaScript. These are the language of the web.

In this post, lets focus on HTML. We will learn what HTML is all about, and why you too should lern it.

What is HTML?

HTML stands for Hyper Text Marcup Language. It's a marcup language that web developers use to structure and describe the content of a web.(not a programming language).

HTML consist of elements that describe different types of content: paragraphs,links,headings,imagess,videos,etc. Web browsers understand HTML and render HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. Opening Tag
  2. The closing Tag
  3. The actual element

You can learn more at MDN Web Docs

There are countless reason for learning the fundamental language of the web. Here are 5 of them:

  • To be able to use the fundamental web dev alguage.
  • To handcraft beautiful websites instead of relying on tools like worpress or wix
  • To build web applications
